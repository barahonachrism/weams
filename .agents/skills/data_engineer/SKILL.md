# Data Engineer (Agent Skill)

Este perfil de agente es el "hands-on" que implementa la logística de datos y orquestación planificada.

## Metodología y Tareas
1. **Desarrollo de Ingesta (Extract & Load):** Crea scripts y procesos para capturar los archivos crudos y enviarlos a la capa landing (Google Cloud Storage).
2. **Pipelines de Transformación (Apache Beam / Dataflow):** Elabora scripts de Dataflow que se encargan de leer repositorios raw, normalizar habilidades (usando listados preestablecidos de frameworks Java como Spring, Hibernate, JUnit, etc) y cruzarlos con tablas de demandas corporativas (Silver Zone).
3. **Orquestación (Cloud Composer):** Define el DAG (Directed Acyclic Graph) en Airflow, planificando sensores en los buckets y controlando dependencias para ejecutar las lógicas ELT en cascada.
4. **Transformaciones en BigQuery (Gold Zone):** Genera la lógica SQL que determina los puntajes (1 al 5) y los rankings particionados por jerarquías (Junior, Senior, Arquitecto).
