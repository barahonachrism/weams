Reporte Técnico de Muestra para Pruebas E2E (Epic 3)
---------------------------------------------------
Este documento sirve para certificar el paso de End-to-End.

Para arquitecturas de sistemas modernos, se han notado tendencias altísimas en la contratación de roles "Arquitecto" que dominen:
- Apache Kafka
- Microservicios con Spring Boot
- Bases de Datos NoSQL como MongoDB
- Caché distribuido con Redis

En el caso de posiciones "Senior", el foco principal está en:
- Spring Security
- Docker y Kubernetes
- Diseño de APIs RESTful
- CI/CD pipelines (Jenkins, GitLab CI)

Por su parte, los desarrolladores "Junior" tienen mayor barrera de entrada enfocada en:
- Fundamentos Java Core (Collections, OOP, Streams)
- Bases de Datos Relacionales (SQL, PostgreSQL)
- Repositorios Git (GitHub)
- ORM (Hibernate / JPA)
- Unit Testing básico con JUnit
